# ANALYSIS_SYSTEM.md

## Purpose

The blood analysis system is one of the core components of Health Coach.

Its purpose is to identify physiological factors that may influence:

- Energy levels
- Productivity
- Motivation
- Cognitive performance
- Emotional well-being
- Recovery capacity
- Hormonal balance
- Overall health optimization

The system does not provide medical diagnoses.

The system provides educational insights, personalized recommendations, and actionable health improvement plans.

---

# Analysis Packages

The application supports predefined laboratory analysis packages.

Users may:

- Upload complete laboratory packages
- Upload individual biomarkers
- Compare results over time
- Receive AI-generated insights
- Track biological improvements
- Receive personalized action plans

# Biomarker Intelligence Layer

Health Coach does not interpret biomarkers only as isolated values.

The AI system groups biomarkers into biological systems and calculates system-level health scores.

The goal is to help users understand causes rather than isolated laboratory values.

# AI Summary Layer

After every analysis, the AI must answer five key questions:

1. What is currently limiting the user's energy?
2. What is currently limiting the user's productivity?
3. What is currently limiting the user's motivation?
4. Which biological systems require attention first?
5. Which actions will create the highest positive impact?

The user should always receive actions, not laboratory numbers.

---

# User Experience Principle

The application should prioritize clarity over complexity.

Users should primarily see:

- Health Scores
- Key Problems
- Recommended Actions
- Progress Tracking

Raw laboratory values should remain available but secondary.

---

# Blood Test Preparation Guide

The application shall provide a preparation checklist before testing.

Recommendations:

- Avoid excessive physical activity the day before testing.
- Avoid emotional stress before testing.
- Avoid overeating.
- Maintain consistent testing conditions.
- Sleep adequately before testing.
- Perform repeated testing at approximately the same time of day.

Purpose:

Increase reliability and comparability of laboratory results.
